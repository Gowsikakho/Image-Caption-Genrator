# Image Caption Generator

A Flask-based web application that generates captions for images using AI.

## Features

- AI-powered image caption generation
- Drag and drop image upload interface
- Caption history tracking
- Elegant golden-themed UI

## Installation

1. Clone the repository
2. Install dependencies using `pip install -r requirements.txt`
3. Run the application using `python app.py`

## Usage

Upload an image using the web interface to generate captions automatically.
